== Description ==
Change the WooCommerce cart 'Continue Shopping' link to a link to the product category instead of the product itself

== Installation ==

1. You have a couple of options:
	* Go to Plugins -> Add New and search for "WooCommerce Continue shopping". Once found, click "Install".
	* Download the folder from Wordpress.org and unzip the folder. Then upload via Plugins -> Add New -> Upload.
2. Activate the plugin through the 'Plugins' menu in WordPress.
